Change the name of dbinfo.example.php file to dbinfo.php 
And open the dbinfo.php file and find the following lines:

# Information for connecting to database
$dbinfo_hostname = "";      // Host
$dbinfo_port = "";          // Port
$dbinfo_username = "";      // Username
$dbinfo_password = "";      // Password
$dbinfo_charset = "utf8";   // Charset
$dbinfo_dbname = "";        // Database Name
$dbinfo_tablename = "chatlog"; // Table name (cvar sm_chat_log_table)

In quotes, enter the corresponding database data - those that you specified in the databases.cfg file on your server.
